Oque são tags HTML?

Tags HTML são rótulos utilizados para marcar e organizar o conteúdo de uma página web. Elas informam ao navegador como exibir cada parte do conteúdo, como títulos, parágrafos, imagens e links. Sempre são escritas entre os sinais < e>, como < p> para parágrafos ou < h1> para títulos principais. Essas tags são essenciais para a estrutura e a formatação de sites.

2. Principais tags utilizadas

Algumas tags são muito utilizadas no desenvolvimento de sites por sua função essencial. Aqui estão algumas delas:

  • html: Indica o início de um documento HTML.
  • head: Contém informações sobre a página, como título e links de estilo.
  • body: É onde fica todo o conteúdo visível, como textos, imagens e links
  • h1 até h6: Títulos de diferentes níveis, do mais importante (h1) ao menos importante (h6).
  • p: Define um parágrafo.
  • a href: Cria um link clicável.
  • img: Exibe uma imagem.

3. Diferença entre tags de abertura e fechamento

Voltar

A maioria das tags HTML é formada por um par: uma tag de abertura e outra de fechamento, como < p> e < /p>, que envolvem o conteúdo a ser exibido. A tag de fechamento é parecida com a de abertura, mas tem uma barra. Já algumas tags, como < img>, são auto-fechadas, pois não possuem conteúdo interno, dispensando a tag de encerramento. Essas estruturas ajudam a organizar corretamente os elementos da página.

4. Listas ordenadas e não ordenadas

< ol> e < ul> no HTML: qual a diferença? No HTML, usamos duas tags principais pra criar listas: < ol> e < ul>. A diferença é bem simples:

  • < ol> (ordered list) cria listas ordenadas, ou seja, com números. Ideal pra mostrar uma sequência, tipo um passo a passo.
  • < ul> (unordered list) faz listas não ordenadas, com bolinhas (●) ou outros símbolos. Boa pra listar itens soltos, como ingredientes ou tópicos.
E para deixar em ordem de lista não basta apenas usar o ol e ul tem que adicionar o < li> conhecido como item da lista assim se não usar o < li> a lista não tem contéudo.

4. HTML no dia a dia

O HTML está presente em praticamente tudo em que acessamos na internet no dia a dia, lojas virtuais, os próprios blogs como este, sites de noticia, redes sociais e etc. Sempre que abrimos um navegador notamos que os textos, botoes e imagens que vemos tem html envolvido, mesmo sem percerber, usamos html o tempo todo. Para quem trabalha com comunicação, marketing ou tecnologia, entender o HTML facilita todas as tarefas de conteúdo, corrigir os erros e criar páginas eficazes.
